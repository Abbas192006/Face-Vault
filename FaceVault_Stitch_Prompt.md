# Stitch Design Prompt - FaceVault AI

Design a futuristic AI-powered face search platform called FaceVault AI.

The product allows users to upload one face image and instantly find matching photos from a local library of thousands of images.

Create a completely original design. Do not imitate Google Photos, Apple Photos, Dropbox, Notion, Linear, or any existing product.

## Design Language
- Neo-Cinematic Glass
- Premium SaaS quality
- Dark mode first
- Depth-heavy interface
- Floating panels
- Soft glass morphism
- Dynamic lighting
- Animated mesh gradient background
- Subtle particle effects
- Smooth 120fps feel

## Motion Requirements
- Framer Motion style interactions
- Micro-interactions everywhere
- Animated page transitions
- Elastic hover effects
- Card lift and tilt
- Animated loading skeletons
- Blur-to-sharp image reveals
- Progressive content appearance
- Magnetic floating buttons

## Layout
- Left navigation rail
- Central AI workspace
- Right insights panel

## Sections
1. AI Search Workspace
2. Face Upload Area
3. Search Results Gallery
4. Similarity Slider
5. Statistics Dashboard
6. Indexed Libraries
7. Recent Searches
8. Face Clusters

## Visual Features
- Interactive glowing search orb
- Animated scan lines
- Live indexing visualization
- Glass cards
- Dynamic reflections
- Floating image previews
- Animated confidence indicators

## Avoid
- Generic dashboards
- Bootstrap styling
- Material Design clones
- Template-looking layouts

Goal: A premium AI product that feels like a 2030 startup platform.
